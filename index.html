Capstone

NLP system for classifying Indonesian financial transactions. Built an LSTM model with a custom Attention Layer (TensorFlow/Keras) trained on 37,000 records across 9 spending categories, achieving 96% validation accuracy. Added an LLM preprocessing layer (Groq API + LLaMA 3.3) to normalize informal user input before classification. Built a full OCR pipeline using OpenCV, EasyOCR and LLM parsing for receipt scanning. Deployed the model via FastAPI and hosted on Hugging Face Spaces as part of a six-person team.
